---
title: my.hideToast
description: Hide the currently displayed toast.
---

Use this API to manually hide a toast before its duration expires.

## Parameters

This API does not require any parameters.

## Code Example

```javascript
my.showToast({
  content: 'Processing...',
  duration: 10000
});

// Hide early when done
setTimeout(() => {
  my.hideToast();
}, 2000);
```
