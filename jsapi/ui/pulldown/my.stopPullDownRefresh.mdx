---
title: my.stopPullDownRefresh
description: Stop the pull-down refresh animation.
---

Use this API to stop the pull-down refresh animation.

## Parameters

This API does not require any parameters.

## Code Example

```javascript
Page({
  onPullDownRefresh() {
    this.fetchData().then(() => {
      my.stopPullDownRefresh();
    });
  }
});
```
